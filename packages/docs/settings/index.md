# Настройки

В этом разделе описаны базовые настройки **Core UI**, которые используются для конфигурации библиотеки в приложении.

Сюда относятся глобальные параметры, влияющие на внешний вид и поведение компонентов, а также общие настройки, используемые на уровне всей UI-системы.

---

## Что находится в этом разделе

Раздел настроек включает параметры, которые применяются не к одному конкретному компоненту, а ко всей библиотеке в целом.

Например:

- глобальные CSS-переменные
- базовые визуальные значения
- настройки темизации
- инфраструктурные параметры библиотеки

---

## Назначение настроек

Настройки позволяют централизованно управлять общими параметрами интерфейса и поддерживать единый подход во всех продуктах компании.

Это упрощает:

- кастомизацию внешнего вида
- поддержку консистентности между проектами
- масштабирование UI-системы
- развитие библиотеки без дублирования локальных решений

---

## Подход

Core UI использует простой и предсказуемый подход к настройке.

Базовые параметры выносятся на глобальный уровень и затем используются внутри компонентов через локальные CSS-переменные и внутренние механизмы библиотеки.

Такой подход позволяет сохранять гибкость настройки без усложнения API компонентов.
