# CApp

`CApp` — компонент библиотеки Vueland UI.

Эта страница сохраняет маршрут исходной документации и описывает назначение компонента, чтобы навигация не вела на несуществующие страницы.

## Использование

```vue
<capp />
```

## Раздел в разработке

Подробное описание API компонента, props, slots, events и CSS-переменных можно расширять по мере развития библиотеки.
