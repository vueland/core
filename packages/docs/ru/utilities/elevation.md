# Elevation

Набор utility-классов Vueland UI для задачи `elevation`.

## Пример

```html
<div class="elevation">Content</div>
```

## Назначение

Эти классы помогают быстро решать типовые задачи интерфейса без написания дополнительного CSS. Используйте их для небольших локальных правок layout, размеров, текста и визуальных состояний.
