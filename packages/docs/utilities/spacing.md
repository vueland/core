# Утилиты отступов (Spacing)

Набор утилитарных классов для управления внешними (`margin`), внутренними (`padding`) отступами и расстоянием между элементами (`gap`).

Эти классы позволяют быстро управлять layout без написания дополнительного CSS.

---

## Шкала отступов

Все значения основаны на единой шкале.

```
шаг = 4px
диапазон = 0 → 20
```

Формула:

```
значение = индекс × 4px
```

Примеры:

| Класс | Значение |
|------|------|
| `.pa-1` | `4px` |
| `.pa-2` | `8px` |
| `.pa-4` | `16px` |
| `.pa-8` | `32px` |

---

## Margin (внешние отступы)

Классы для управления `margin`.

| Класс | CSS |
|-----|-----|
| `.ma-{n}` | `margin` |
| `.mt-{n}` | `margin-top` |
| `.mr-{n}` | `margin-right` |
| `.mb-{n}` | `margin-bottom` |
| `.ml-{n}` | `margin-left` |
| `.mx-{n}` | `margin-left + margin-right` |
| `.my-{n}` | `margin-top + margin-bottom` |

### Пример

```html

<div class="ma-4">
  Блок с внешним отступом
</div>

<div class="mt-6 mb-4">
  Блок с вертикальными отступами
</div>
```

---

## Padding (внутренние отступы)

Классы для управления `padding`.

| Класс | CSS |
|-----|-----|
| `.pa-{n}` | `padding` |
| `.pt-{n}` | `padding-top` |
| `.pr-{n}` | `padding-right` |
| `.pb-{n}` | `padding-bottom` |
| `.pl-{n}` | `padding-left` |
| `.px-{n}` | `padding-left + padding-right` |
| `.py-{n}` | `padding-top + padding-bottom` |

### Пример

```html

<div class="pa-4">
  Блок с внутренним отступом
</div>

<div class="px-6 py-3">
  Блок с горизонтальными и вертикальными отступами
</div>
```

---

## Gap (расстояние между элементами)

Классы для управления расстоянием между элементами в `flex` и `grid` контейнерах.

| Класс | CSS |
|-----|-----|
| `.gap-{n}` | `gap` |
| `.gap-x-{n}` | `column-gap` |
| `.gap-y-{n}` | `row-gap` |

### Пример

```html

<div class="d-flex gap-4">
  <div>Item</div>
  <div>Item</div>
  <div>Item</div>
</div>
```

---

## Адаптивные классы (Responsive)

Все утилиты поддерживают адаптивные модификаторы.

Формат:

```
{breakpoint}:{utility}
```

Примеры:

| Класс | Описание |
|-----|-----|
| `.md:pa-4` | padding применяется начиная с `md` |
| `.lg:mx-6` | горизонтальные margin на `lg` |
| `.xl:gap-8` | gap на `xl` |

### Пример

```html

<div class="pa-2 md:pa-4 lg:pa-8">
  Адаптивные отступы
</div>
```

---

## Брейкпоинты

Значения берутся из карты:

```scss
$grid-breakpoints
```

Пример:

| Брейкпоинт | Минимальная ширина |
|------------|--------------------|
| `sm`       | `576px`            |
| `md`       | `768px`            |
| `lg`       | `992px`            |
| `xl`       | `1200px`           |
| `xxl`      | `1920px`           |

---

## Полный пример

```html

<div class="pa-6">
  <div class="mb-4">
    Заголовок
  </div>
  <div class="d-flex gap-4">
    <div class="pa-4">Карточка</div>
    <div class="pa-4">Карточка</div>
    <div class="pa-4">Карточка</div>
  </div>
</div>
```

---

# Итог

Утилиты отступов позволяют:

- управлять `margin` и `padding`
- быстро выстраивать layout
- использовать единую шкалу отступов
- применять адаптивные модификаторы

Все классы используют `!important`, чтобы гарантировать приоритет над обычными стилями.
