# Утилиты размера шрифта

Набор утилитарных классов для управления размером текста через `font-size`.

Утилиты позволяют быстро применять фиксированную типографическую шкалу без написания дополнительного CSS.

Все классы используют `!important`.

---

## Шкала размеров

Размеры текста задаются через карту `$font-sizes`.

| Ключ | Размер |
|------|--------|
| `xs` | `12px` |
| `sm` | `14px` |
| `base` | `16px` |
| `lg` | `18px` |
| `xl` | `20px` |
| `2xl` | `24px` |
| `3xl` | `32px` |

---

## Классы

Для каждого значения генерируется класс формата:

```text
.fs-{name}
```

| Класс | CSS |
|------|------|
| `.fs-xs` | `font-size: 12px` |
| `.fs-sm` | `font-size: 14px` |
| `.fs-base` | `font-size: 16px` |
| `.fs-lg` | `font-size: 18px` |
| `.fs-xl` | `font-size: 20px` |
| `.fs-2xl` | `font-size: 24px` |
| `.fs-3xl` | `font-size: 32px` |

---

## Визуальные примеры

<div class="d-flex flex-col gap-3">
  <div class="fs-xs">Текст размера xs — 12px</div>
  <div class="fs-sm">Текст размера sm — 14px</div>
  <div class="fs-base">Текст размера base — 16px</div>
  <div class="fs-lg">Текст размера lg — 18px</div>
  <div class="fs-xl">Текст размера xl — 20px</div>
  <div class="fs-2xl">Текст размера 2xl — 24px</div>
  <div class="fs-3xl">Текст размера 3xl — 32px</div>
</div>

---

### Пример использования

```html
<p class="fs-sm">Вспомогательный текст</p>
<p class="fs-base">Основной текст</p>
<h3 class="fs-xl">Заголовок секции</h3>
<h1 class="fs-3xl">Крупный заголовок</h1>
```

---

## Адаптивные модификаторы

Все утилиты размера шрифта поддерживают responsive версии.

Формат:

```text
{breakpoint}:fs-{name}
```

Примеры:

| Класс | Описание |
|------|----------|
| `.md:fs-lg` | размер `18px` начиная с `md` |
| `.lg:fs-xl` | размер `20px` начиная с `lg` |
| `.xl:fs-2xl` | размер `24px` начиная с `xl` |

### Пример

```html
<h2 class="fs-lg md:fs-xl lg:fs-2xl">
  Адаптивный заголовок
</h2>
```

---

## Брейкпоинты

Значения берутся из карты:

```scss
$grid-breakpoints
```

| Брейкпоинт | Минимальная ширина |
|------------|--------------------|
| `sm`       | `576px`            |
| `md`       | `768px`            |
| `lg`       | `992px`            |
| `xl`       | `1200px`           |
| `xxl`      | `1920px`           |

---

## Рекомендации

Используйте типографическую шкалу последовательно:

- `fs-xs`, `fs-sm` — вспомогательный и вторичный текст
- `fs-base` — основной текст по умолчанию
- `fs-lg`, `fs-xl` — акцентный текст и подзаголовки
- `fs-2xl`, `fs-3xl` — заголовки

Для адаптивной типографики комбинируйте базовый размер и breakpoint-модификаторы.

---

## Итог

Утилиты размера шрифта позволяют:

- быстро применять единую типографическую шкалу
- избегать произвольных значений в шаблонах
- строить адаптивную типографику
- поддерживать единообразие интерфейса
