# Utility Classes

Утилитарные классы — это небольшой набор вспомогательных CSS-классов для частых layout- и typographic-задач.

Они не заменяют компонентный подход и не превращают библиотеку в Tailwind-style фреймворк. Их задача — закрывать самые частые сценарии:

- подбор цвета
- позиционирование
- отображение
- flex/grid layout
- отступы
- размеры
- overflow
- прозрачность
- текстовые helper-классы

## Принципы

### Локальный и ограниченный набор

Мы добавляем только те utility-классы, которые реально часто используются в интерфейсах.

### Предсказуемый нейминг

Названия должны быть короткими, очевидными и единообразными.

### Utility — не замена компонентам

Если набор классов начинает описывать целый UI-блок, лучше вынести это в отдельный компонент или semantically named class.
