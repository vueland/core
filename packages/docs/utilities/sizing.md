# Утилиты размеров (Sizing)

Набор утилитарных классов для управления размерами элементов.

Позволяют быстро задавать:

- ширину (`width`)
- высоту (`height`)
- минимальные размеры (`min-width`, `min-height`)

Все классы используют `!important`.

---

## Доступные классы

| Класс | CSS |
|------|------|
| `.w-100` | `width: 100%` |
| `.h-100` | `height: 100%` |
| `.w-auto` | `width: auto` |
| `.h-auto` | `height: auto` |
| `.min-w-0` | `min-width: 0` |
| `.min-h-0` | `min-height: 0` |

---

# Ширина

## `.w-100`

Задаёт ширину элемента равную **100% родительского контейнера**.

<div style="border:1px dashed var(--vp-c-divider)" class="pa-4">
<div class="w-100 pa-2 elevation-1">
width: 100%
</div>
</div>

```html

<div class="w-100">
  Контент
</div>
```

---

## `.w-auto`

Ширина элемента определяется **его содержимым**.

```html

<div class="w-auto">
  Контент
</div>
```

---

# Высота

## `.h-100`

Задаёт высоту элемента равную **100% высоты родителя**.

<div style="height:120px;border:1px dashed var(--vp-c-divider)" class="pa-4">
<div class="h-100 pa-2 elevation-1">
height: 100%
</div>
</div>

```html

<div class="h-100">
  Контент
</div>
```

---

## `.h-auto`

Высота элемента определяется содержимым.

```html

<div class="h-auto">
  Контент
</div>
```

---

# Минимальные размеры

## `.min-w-0`

Разрешает элементу **сжиматься меньше ширины содержимого**.

Это особенно полезно в `flex` контейнерах.

```html

<div class="min-w-0">
  Контент
</div>
```

---

## `.min-h-0`

Разрешает элементу **сжиматься по высоте**.

Используется в `flex` и `grid` layout.

```html

<div class="min-h-0">
  Контент
</div>
```

---

# Частый кейс: flex layout

В flex-контейнерах элементы иногда **не сжимаются из-за min-width**.

Решение — использовать `.min-w-0`.

```html

<div class="d-flex">
  <div class="min-w-0">
    Очень длинный текст который должен переноситься
  </div>
</div>
```

---

# Пример интерфейса

```html

<div class="d-flex gap-4">
  <div class="w-100 pa-4 elevation-1">
    Основной контент
  </div>
  <div class="w-auto pa-4 elevation-1">
    Sidebar
  </div>
</div>
```

---

# Рекомендации

Используйте:

- `.w-100` для растягивания блока
- `.h-100` для заполнения контейнера
- `.w-auto` для естественного размера
- `.min-w-0` внутри flex layout
- `.min-h-0` внутри grid/flex контейнеров

---

# Итог

Утилиты размеров позволяют:

- быстро управлять шириной и высотой элементов
- строить адаптивные layout
- корректно работать с flex и grid контейнерами
- избегать неожиданных ограничений `min-width`
